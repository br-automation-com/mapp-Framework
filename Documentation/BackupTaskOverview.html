mapp Backup Framework - Task Overview

The BackupMgt task contains all of the provided Framework code for mapp Backup. The chart below provides a description for the main task and each action file.


Filename

Type

Description

BackupMgt.st

Main task code

General mapp function block handling.

This task contains all of the necessary code to create, restore, and delete a backup of the program.

HMIActions.st

Action

Supporting code for HMI functionality.

Loads and saves the backup configuration.

File manager handling.

ChangeConfiguration.st

Action

Contains the programming to modify the backup configuration at runtime. Modifications to this action are typically not necessary.
